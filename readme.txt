The aim of this code is to run, analyse, extract info and train three computational models on the temperature dataset provided by
Indian Meteorological Department. All ethical requirements of the data is satisfied. the data along with the metadata can be found at https://drive.google.com/drive/folders/16tK8OPlRUYAovjmEz7lygkG2AIRqiGbN?usp=sharing


TO ensure the working of this code in google colab. Please make sure you download and  upload this data folder provided in the link to your drive
change the path of the initial few blocks to your dataset folder if required(for some reason if you have changed the name). the places where changes are required are marked via comments inside the code so you will not have any issue.

DISCLAIMER

the entire dataset in its extracted form is expected to be around 10GB.
